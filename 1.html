HTML이란 무엇인가?

Hypertext Markup Language (HTML) is the standard markup language for creating web pages and web applications. Web browers receive HTML documents from a web server or from local storage and render them into multimedia web pages. HTML describes the structure of a web page semantically and originally included cues for the apperance of the document.

HTML elements are the building blocks of HTML pages. With HTML constructs, images and other objects, such as interactive forms, may be embedded into the rendered page. It provides a means to create structured documents by denoting structural semantics for text such as headings, paragraphs, lists, links, quotes and other items. HTML elements are delineated by tags, written using angle brackets.

1. TAG의 종류들

  1. h1 ~ h6 : 제목
  2. li : 목록, 목차를 쓸 때 사용하는 태그
  3. u : 밑줄
  4. strong : 강조
  5. br : 줄바꿈 (닫히는 태그 없음)
  6. p : 단락 나누기
  7. ol : ordered list. 번호가 매겨진 리스트. ol 안하면 그냥 dot 으로 표시된 리스트가 됨
  8. ul : unordered list. 이거를 단락 앞뒤에 붙이냐 안붙이냐에 따라 단락의 왼쪽 여백이 생김... 무슨 용도인지는 아직 잘 모르겠다
    9. ul, ol 과 li 태그는 서로 부모자식 태그임!
  9. ⭐️ CSS 에서 태그와 태그 사이의 여백을 나타내는 코드가 있음. 두번째 태그에 style 이라는 HTML의 속성을 쓰면 됨.
    그리고 속성의 값은, CSS라는 HTML과 다른 컴퓨터 언어의 문법이 들어오게 약속되어 있음. 위의 예문 단락에서, 두번째 p 태그.
    margin 이란 태그와 태그사이의 여백. top 은, 두번째 P 태그의 '위쪽'에 여백을 주고 싶은 것이기 때문.
    40px 은, 컴퓨터 상에서 사용하는 단위, 40px 만큼의 여백을 주겠다는 뜻.
  10. p 태그를 통해 어디서부터 어디까지가 한 단락이다, 라는 것을 나타내고,
    부족한 부분은 CSS를 통해 훨씬 더 정교하게 제어할 수 있는 것(단락 간 여백의 크기, 여백의 위치 등)
  11. title : 만든 웹 페이지의 제목을 만들어 줌
  12. meta charset="utf-8" : 문자(character) 세팅(set)을 utf-8로 읽어라, 라는 의미
    13. 편집기에서 한글을 썼을 때 결과물에서 한글이 안나오고 이상하게 나온다면, 우리의 편집기는 밑에 보면 UTF-8 이라는 방식으로 저장된거라, 웹 브라우저가 이걸 같은 양식으로 읽게해야 하기 때문임. 따라서 해당 명령어를 코드들의 가장 위에 써주면 그 밑으로의 코드들은 다 utf-8로 읽는 것임
  13. meta : 항상 head 요소 안에 들어감. 일반적으로 문자 집합, 페이지 설명, 키워드, 문서 작성자 및 뷰포트 설정을 지정하는데 사용
  14. body : 본문임을 나타내는 태그는 body 태그로 묶기로 '약속'함
  15. head : 본문을 설명하는 부분(즉, 제목)은 head 태그로 묶기로 '약속'함
  16. html : head와 body 태그를 유일하게 묶을 수 있는 태그
  17. !doctype html : 관용적으로, html 태그 위에 사용해 해당 문서가 html 문서라고 선언하는 용도
  18. a : Anchor. 정보의 바다에 정박한다는 의미. 이것이 있기때문에 검색엔진이 검색을 할 수 있는 것. '링크'를 표현하는 태그임.
    a href 에서 href 는 a의 속성. h 는 하이퍼 텍스트, 즉 링크. ref 는 '웹 브라우저야, 이 링크는 기 값을 '참조'해' 하는 것.'
    ⭐️ 하이퍼텍스트에서 링크를 새 창에서 열기를 하고 싶은 경우 속성에 target = "_blank" 를 추가하면 됨
    이 링크를 클릭하기 전에, 이 링크 위에 마우스 커서를 올려두면 이 링크에 대한 설명이 나타나는 기능(툴팁)을 추가하고 싶다? 속성에 title = "html5 specification" 같이 "" 안에 설명을 넣으면 됨.
